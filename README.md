# mad-blip
created using: create-react-app, NestJs, Knex, sqlite3, next-knexjs, axios
story written by: jzkime

## How to install
Set up backend
1. cd into /server-side 
2. run `$ npm ci`
3. run `$ npm run start`
4. run `$ npm run resetdb`

Set up frontend
1. cd into /client-side
2. run `$ npm ci`
3. run `$ npm run start`
